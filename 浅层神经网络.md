# 浅层神经网络

## 神经网络表示　neural network Representation

### 双层神经网络
* input layer $a^{[0]}=x$
* hidden layer $a^{[1]}$
* output layer $a^{2} \hat{y}=a^{[i]}$
>计算神经网络的层数，不包含输入层　\
计算过程　\
given input x: \
$z^{[1]}=W^{[1]}+b^{[1]}$ \
$a^{[1]}=\sigma (z^{[1]})$ \
$z^{[2]}=W^{[2]}a^{[1]}+b^{[2]}$ \
$a^{[2]}=\sigma (a^{[2]})$ \
计算过程图　可以看补充课件　神经网络计算过程

### vectorizing across multiple examples (m个训练样本的向量化)
以双层神经网络为例\
$X \in R^{nx*m}$ \
$Z^{[1]}=W^{[1]}X+b^{[1]}$ \
$A^{[1]}= \sigma (Z^{[1]})$ \
$Z^{[2]}=W^{[2]}A^{[1]}+b^{[2]}$ \
$A^{[2]}=\sigma (Z^{[2]})$
